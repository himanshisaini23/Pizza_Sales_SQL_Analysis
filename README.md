# 🍕 Pizza Sales Analysis using SQL

## 📌 Project Overview

This project analyzes a pizza restaurant's sales data using SQL to uncover business insights related to revenue, customer ordering patterns, and product performance. The objective is to answer real-world business questions by writing SQL queries that transform raw transactional data into meaningful insights for decision-making.

---

## 🎯 Objectives

- Analyze overall sales performance.
- Identify the best-selling pizzas and categories.
- Calculate total revenue generated.
- Understand customer ordering behavior.
- Discover sales trends using SQL.

---

## 🗂️ Dataset

The dataset consists of multiple related tables representing a pizza restaurant's sales operations.

- Orders
- Order Details
- Pizzas
- Pizza Types

These tables were joined using SQL to perform comprehensive sales analysis.

---

## 🛠️ SQL Concepts Used

- SELECT Statements
- WHERE Clause
- ORDER BY
- GROUP BY
- HAVING
- Aggregate Functions (SUM, COUNT, AVG)
- INNER JOIN
- LIMIT
- Date & Time Functions
- Subqueries
- Common Business Calculations

---

## 📊 Business Questions Solved

- What is the total number of orders?
- What is the total revenue generated?
- Which pizza generated the highest revenue?
- Which pizza category is most popular?
- Which pizza size is ordered the most?
- What are the top 5 best-selling pizzas?
- What is the average number of pizzas ordered per day?
- Which hours receive the highest number of orders?
- Which pizza categories contribute the most to total sales?

---

## 📈 Key Insights

- Identified the highest revenue-generating pizza products.
- Compared sales across pizza categories and sizes.
- Analyzed customer ordering patterns throughout the day.
- Calculated overall restaurant revenue.
- Ranked products based on sales performance.
